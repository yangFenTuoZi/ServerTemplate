# ServerTemplate

继承ServerTemplate类就可以在Android轻松实现简单的server

自带Logger和FakeContext，可以选择开启

**默认的`onStart()`是在子线程中运行**

**若要切换到主线程，请使用`runOnMainThread(Runnable action)`**
